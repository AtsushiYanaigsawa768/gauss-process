# data/ -- データディレクトリ概要

[English version](../../../data/)

## 概要

`data/` はフレキシブルリンク機構のシステム同定に使用する実験データを格納する。
時系列データ (.mat) と周波数応答データ (.dat) の2種類がある。

## ディレクトリ構造

```
data/
└── sample_data/    10個の .mat ファイル (時系列計測データ)
```

## データの役割

### sample_data/ (時系列データ)

パイプラインの入力として使用する。以下の2つの用途がある:

1. **周波数応答推定の入力**: `frequency_transform/` モジュールが .mat ファイルから FRF を計算
2. **FIR検証の参照データ**: `fir_model/fir_validation.py` が時間領域シミュレーションの精度を評価
